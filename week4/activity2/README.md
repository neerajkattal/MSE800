#  Rectangle Area and Perimeter (OOP Project)

##  Overview

This project is an Object-Oriented Programming (OOP) application that calculates the **area** and **perimeter** of a rectangular piece of land.

The user inputs the length and width, and the program computes the results using a class.

---

##  Files in the Project

### 1. rectangle.py

Contains the `Rectangle` class.

### 2. main.py

Handles user input and displays output.

---

##  Class Description

### Rectangle Class

**Attributes:**

* length
* width

**Methods:**

* calculate_area() → returns area of rectangle
* calculate_perimeter() → returns perimeter of rectangle

---

##  Technologies Used

* Python (OOP concepts)

---

##  How to Run

1. Open terminal
2. Run the program:

   ```bash
   python main.py
   ```
3. Enter length and width when prompted

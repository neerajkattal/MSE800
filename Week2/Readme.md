# OOP Calculator Project

##  Description

This project is a Python-based calculator developed using Object-Oriented Programming (OOP) concepts. It performs basic mathematical operations, factorial calculations, and complex number operations.

##  OOP Concepts Used

### 1. Class

A `Calculator` class is created to encapsulate all operations.

### 2. Methods

The class contains multiple methods:

* `calculate()` → performs arithmetic operations
* `factorial()` → computes factorial using recursion
* `add_complex()` → adds complex numbers
* `multiply_complex()` → multiplies complex numbers

### 3. Functions

Two standalone functions are used:

* `get_numbers()` → handles user input
* `main()` → controls program execution

## Features

* Addition, subtraction, multiplication, division, modulus
* Factorial calculation
* Complex number operations

## ▶How to Run

1. Clone the repository
2. Run the Python file:

   ```bash
   python calculator.py
   ```

## Conclusion

This project demonstrates how OOP improves code organization, reusability, and readability compared to procedural programming.
